# Formulário de Matrícula

Projeto desenvolvido com HTML5 e CSS3 que simula um formulário de matrícula para uma instituição de ensino. O objetivo é praticar a criação de formulários completos, organização de campos, estilização de componentes e construção de interfaces responsivas.

## Acesse o projeto

**Deploy:** https://daniellydev.github.io/formulario_de_matricula/

## Sobre o projeto

Este projeto consiste em uma interface de formulário de matrícula desenvolvida para praticar a criação de formulários utilizando HTML5 e CSS3.

Durante o desenvolvimento foram aplicados conceitos de semântica, agrupamento de informações, estilização de campos de entrada e organização visual da interface, proporcionando uma experiência de preenchimento clara e intuitiva.

## Tecnologias utilizadas

* HTML5
* CSS3
* Git
* GitHub
* GitHub Pages

## Funcionalidades

* Formulário de matrícula completo
* Campos de texto
* Seleção de opções
* Botões de rádio e caixas de seleção
* Upload de arquivos
* Organização das informações em seções
* Layout responsivo

## Conceitos praticados

* Estrutura HTML semântica
* Criação de formulários
* `input`, `label`, `textarea` e `button`
* `select`, `option` e `fieldset`
* Botões de rádio (`radio`)
* Caixas de seleção (`checkbox`)
* Upload de arquivos (`file`)
* Validação básica de formulários
* Flexbox
* Responsividade
* Organização de componentes com CSS

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/daniellydev/formulario_de_matricula.git
```

2. Acesse a pasta do projeto:

```bash
cd formulario_de_matricula
```

3. Abra o arquivo `index.html` em seu navegador.

* Organizar formulários extensos de forma clara e intuitiva.
